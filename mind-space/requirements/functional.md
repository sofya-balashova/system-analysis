# Функциональные требования

## Управление пользователями
- Регистрация пользователей по email и паролю
- Авторизация зарегистрированных пользователей
- Ролевое разграничение доступа

## Психологи
- Просмотр списка психологов
- Фильтрация по специализации, опыту, цене, рейтингу
- Просмотр профиля психолога

## Контент
- Просмотр списка статей
- Поиск статей по названию
- Чтение статей
- Комментирование статей (для авторизованных пользователей)

## Функционал психолога
- Редактирование профиля
- Публикация статей

## Модерация
- Удаление статей
