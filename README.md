\# System Analysis Projects Repository



Данный репозиторий содержит проекты по системному анализу, разработанные в рамках учебных и исследовательских задач. Все проекты оформлены отдельными папками с собственными README-файлами и документацией.



\## Содержание репозитория



\### 1. Психологический форум

\*\*Цель:\*\* Разработка системы для поиска и взаимодействия с психологами, включая фильтры, чат, заказ консультаций и отзывы.  

\*\*Особенности:\*\*  

\- Система поиска специалистов с фильтрами.  

\- Пользовательский и административный интерфейсы.  

\- История взаимодействий, запись консультаций, уведомления.  

\- Диаграммы: последовательностей, активностей, концептуальная и логическая модели.  



\### 2. Фрилансовая биржа

\*\*Цель:\*\* Платформа для поиска и выполнения заказов фриланс-разработчиками и другими специалистами.  

\*\*Особенности:\*\*  

\- Профили пользователей с рейтингами, портфолио и достижениями.  

\- Управление заказами, отклики, чат, безопасные платежи.  

\- Подписки, премиум-функции, уведомления.  

\- Технологии: Spring Boot, Spring Security, Spring Cloud, RabbitMQ, PostgreSQL, MongoDB, Redis, Minio.  

\- Диаграммы: Use Case, UML, IDEF1X, секвенс- и активити-диаграммы.  



\### 3. Групповые поездки на основе событий (Uber Feature)

\*\*Цель:\*\* Фича для объединения пользователей, направляющихся на одно и то же мероприятие, в совместные поездки с динамическими скидками.  

\*\*Особенности:\*\*  

\- Интеграция с внешними сервисами событий.  

\- Формирование групп, построение маршрута, уведомления пассажиров и водителей.  

\- Система скидок и оптимизации распределения поездок.  

\- Диаграммы: Use Case, BPMN, концептуальная и логическая модели, база данных.
